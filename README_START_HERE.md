## üÜò TROUBLESHOOTING

### Common Issues & Solutions:

**Issue: Signs not being captured**
```
Solution:
1. Check lighting (make it VERY bright)
2. Use plain white wall background
3. Hold gesture 2 full seconds
4. Keep hand perfectly still
5. Make sure entire hand is in frame
```

**Issue: Wrong gesture detected**
```
Solution:
1. Check "Visual Signs" for correct hand shape
2. Make more distinct finger positions
3. Spread fingers wider/clearer
4. Better lighting helps accuracy
5. Refer to ISL reference guide
```

**Issue: Low confidence scores**
```
Solution:
1. IMPROVE LIGHTING (most common fix)
2. Use plainer background
3. Hold longer (2-3 seconds)
4. Make clearer gestures
5. Move to better lit area
```

**Issue: "No hand detected" message**
```
Solution:
1. Show entire hand to camera
2. Move to brighter area
3. Use plain wall background
4. Check camera permissions
5. Refresh browser
```
